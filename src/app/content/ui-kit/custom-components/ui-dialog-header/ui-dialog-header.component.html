Dialog Header Component


Purpose :


The dialog header component is used for standart header of a dialog with icons and a X for close.



Usage :


Custom element: <dialog-header></dialog-header>


To use the dialog header properly you have to pass some parameters to it's properties.


  • 1. [headerTitle] (string) - This property is used to set the title of the dialog.

  • 2. [icon] (string) - This property is used to set the icon in the dialog header.

  • 3. [dialogRef] (MatDialogRef<any>) - This property is used to set the dialogRef for the close to work.

  • 4. [data] (any) - This property is used to set the data to be passed on close if needed.




Example : 


            Code used : 

                <dialog-header headerTitle="Title" [dialogRef]="dialogRef"></dialog-header>
