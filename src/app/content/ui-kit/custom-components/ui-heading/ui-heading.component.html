Heading Component


Purpose :


The heading component is used for standart colored headings.



Usage :


Custom element: <heading></heading>


To use the heading properly you have to pass some parameters to it's properties.


  • 1. [rgbaColor] (string) - This property is used to set the color of the heading.

  • 2. [hTitle] (string) - This property is used to set the title of the heading.

  • 3. [hSubtitle] (string) - This property is used to set the subtitle of the heading.

  • 4. [componentClass] (string) - This property is used to set the classes of the heading.

  • 5. [hStyle] (string) - This property is used to set the orientation of the heading.




Example : 


            Code used : 

                <heading hTitle="Heading Title" rgbaColor="rgba(31, 205, 147,1)"></heading>
