Slider Component


Purpose :


The slider component is used for displaying a slider input.



Usage :


Custom element: <slider-input></slider-input>


To use the slider properly you have to pass some parameters to it's properties.


  • 1. [value] (string) - This property is used to set the value of the slider.

  • 2. [min] (number) - This property is used to set the min of the slider.

  • 3. [max] (number) - This property is used to set slider of the slider.

  • 4. [label] (string) - This property is used to set the label of the slider.

  • 5. [sliderValueSymbol] (string) - This property is used to set the balue of the slider.

  • 6. [step] (number) - This property is used to set the step of the slider.

  • 7. [isDisabled] (boolean) - This property is used to set the isDisabled of the slider.

    • isDisabled



Example : 


            Code used : 

                <slider [isDisabled]="false" sliderValueSymbol="%" label="current-account" step="1" [min]="0" [max]="100">
                </slider>

                <slider [isDisabled]="true" sliderValueSymbol="%" label="current-account" step="1" [min]="0" [max]="100">
                </slider>
