Slider Component


Purpose :

The slider button is used as a toggle-able on/off switch.


Usage:

Custom element: <slider-button></slider-button>


  • 1. [formControlLabel] (string) - Used to set the for and id attributes of the input. This allows you to associate the slider with a <label></label> element.

  • 2. (toggleButton) ([string, boolean]) - Event emitter that emits each time you toggle the button. Emits an array with the formControlLabel and the isChecked boolean


Example:


Code used:


<label for="example-1" class="mb-5"><b>Toggle label 1</b></label>
<slider-button formControlLabel="example-1" (toggleButton)="onToggleButton($event)"></slider-button>
<br>
<label for="example-2" class="mb-5"><b>Toggle label 2</b></label>
<slider-button formControlLabel="example-1" (toggleButton)="onToggleButton($event)"></slider-button>
