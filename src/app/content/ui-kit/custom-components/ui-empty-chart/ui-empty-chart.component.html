Empty Chart Component


Purpose :


The empty chart component is used for standart empty chart.



Usage :


Custom element: <empty-chart></empty-chart>


To use the empty chart properly you have to pass some parameters to it's properties.


  • 1. [componentHeight] (number) - This property is used to set the height of the chart.




Example : 


            Code used : 

                <empty-chart componentHeight="340"></empty-chart>
