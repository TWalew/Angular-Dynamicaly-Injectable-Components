Confirm Dialog Component


Purpose :

The confirm dialog component is used whenever we want a user to confirm an action.


Usage:

To use the dialog modal properly you have to pass some properties to the data object of MatDialog


  • 1. title (string) - Used for the title of the dialog. Can be a translation string

  • 2. text (string) Used for the additional text below the title.

  • 3. confirmButtonText (string) Used for the text of the confirm button. Can be a translation string.

  • 4. withCancelButton (boolean) If true, it will show a cancel button for the modal.


In your component, inject the MatDialog component:

constructor(private dialog: MatDialog) { }

Then, pass an object to the data property of MatDialog that corresponds to the ConfirmDialogInterface 


            public openConfirmDialog() {
               const dialogRef = this.dialog.open(ConfirmDialogComponent, {.
                        width: '500px',
                        panelClass: 'logged-in-dialog'
                        data: {
                            title: 'Confirm Dialog',
                            text: 'Do you confirm this action?',
                            confirmButtonText: 'COMMON.CONFIRM',
                            withCancelButton: true,
                        } as ConfirmDialogInterface,
                });
            }

To handle the click event, you can subscribe to the afterClosed Observable of the dialogRef

            if (event === 'action') {

            }

            if (event === 'cancel') {

            }
        });
