Progress Bar Component


Purpose :


The progress bar component is used for displaying a progress bar.



Usage :


Custom element: <progress-bar></progress-bar>


To use the progress bar properly you have to pass some parameters to it's properties.


  • 1. [progressBarLabel] (string) - This property is used to set the label of the progress bar.

  • 2. [progress] (number) - This property is used to set the progress.

  • 3. [useValue] (boolean) - This property is used to set if the value is used or not.

  • 4. [progressValue] (any) - This property is used to set the value of the progress.

  • 5. [progressSymbol] (string) - This property is used to set the symbol of the progress.

  • 6. [progressClass] (string) - This property is used to set the class of the progress.

  • 7. [progressBarClass] (string) - This property is used to set the class of the progress bar.




Example : 


            Code used : 

                <progress-bar [useValue]="false" progressClass="my-bonuses-progress mt-2"
                progressBarClass="my-bonuses-progress-bar my-bonuses-warn-bar" [progress]="70"></progress-bar>
