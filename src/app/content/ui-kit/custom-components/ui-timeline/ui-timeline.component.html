Timeline Component


Purpose :


The timeline component is used for displaying a timeline.



Usage :


Custom element: <timeline></timeline>


To use the timeline properly you have to pass some parameters to it's properties.


  • 1. [timelineData] (TimelineInterface[]) - This property is used to set the timeline data array.

  • 2. [currentSelectedIndex] (number) - This property is used to set the current day in the timeline.




Example : 


            Code used : 

                <timeline [currentSelectedIndex]="currentDay" [timelineData]="topPerformances"></timeline>
