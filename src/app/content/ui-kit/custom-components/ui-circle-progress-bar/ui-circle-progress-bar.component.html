Circle Progress Bar Component


Purpose :


The circle progress bar component is used to display a progress circle svg with icon.



Usage :


Custom element: <circle-progress-bar></circle-progress-bar>


To use the circle progress bar properly you have to pass some parameters to it's properties.


  • 1. [circleColor] (string) - This property is used to set the color of the bar and icon.

  • 2. [percent] (number) - This property is used to set the percent the progress is at.

  • 3. [icon] (string) - This property is used to set the icon that will be displayed inside the circle and uses font awesome icons.

  • 4. [componentClass] (string) - This property is used to set the classes of the circle progress bar component.




Example : 


            Code used : 
            <circle-progress-bar [percent]="70" componentClass="m-auto" icon="car" circleColor="rgb(31, 205, 148)">
            </circle-progress-bar>
