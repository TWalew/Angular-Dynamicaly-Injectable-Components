Widget Loader Component


Purpose :

The widget loader is used to simulate the loading state inside widgets.


Usage:

Custom element: <widget-loader></widget-loader>


  • 1. [isLoading] (boolean) - Used to set the loading state of the widget loader.

  • 2. [purpose] (string) - Event emitter that emits each time you toggle the button. Emits an array with the formControlLabel and the isChecked boolean



Usage 1 :

You can use the LoadingService and start/stop the loader with the purpose property


Inject into constructor LoadingService.

constructor(private loadingService: LoaderService)


In your fetch data function.

this.loadingService.start('your-purpose');
this.yourApiService.getSomething()
.subscribe(something =>
this.loadingService.stop('your-purpose'); );


Usage 2 :

You can use the isLoading and pass a boolean in order to change the loading state.

Example:


Code used:


<div class="widget-wrapper w-25 h-25">
<h5>Example Loading</h5>
<widget-loader [purpose]="purpose"></widget-loader>
</div>

Example Loading

Example:


Code used:


<div class="widget-wrapper w-25 h-25">
<h5>Example Loaded</h5>
<widget-loader [isLoading]="false"></widget-loader>
</div>
