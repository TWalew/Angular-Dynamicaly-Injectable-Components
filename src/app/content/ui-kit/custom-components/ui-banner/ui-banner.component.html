Banner Component


Purpose :


The banner component is used to give to the user information and some action with events.



Usage :


Custom element: <banner></banner>


To use the banner properly you have to pass some parameters to it's propertyes.


  • 1. [title] (string) - This property is used to set the title of the banner.

  • 2. [bannerActionsTemplate] (TemplateRef) - This property is used to set template of our actions.


Example : 


            Code used : 

            <banner [title]="'Test Page - Test'" [bannerActionsTemplate]="bannerActions"></banner>

    
            <ng-template #bannerActions>
                <fa-icon [icon]="['fas', 'user-secret']"></fa-icon>  <span class="banner-action-title">Impersonate</span> </button>

                <button (click)="onBlock()"> <fa-icon [icon]="['fas', 'user-slash']"></fa-icon> <span class="banner-action-title">Block</span> </button>

                <button (click)="onBan()"> <fa-icon [icon]="['fas', 'user-shield']"></fa-icon> <span class="banner-action-title">Ban</span>  </button>
            </ng-template>
