Loader Button Component


Purpose :


The loader button component is used to simulate loading state for the button while the request is finished.



Usage :


Custom element: <loader-button></loader-button>


To use the loader button properly you have to pass some parameters to it's properties.


  • 1. [butonTitle] (string) - This property is used to set the text inside the button.

  • 2. [componentClass] (string) - This property is used to set the css classes which the button will have.

  • 3. [spinner] (boolean) - This property is used to set the template of the button while in loading state. When = true the button has a spinner, when false it has the text Loading... .

  • 4. [type] (string) - This property is used to set the type of the button element.

  • 5. [purpose] (string) - This property is used to set the purpose of the button so the service knows which button's loading state you want to start / stop.

  • 6. (btnClick) (method) - This property is used to emit an event when the button is clicked so it can start the method you passed to it.


IMPORTANT!


In case you want the button to start loading you have to use the Loader Service.

Example : 



            Inject into constructor LoadingService.

            constructor(private loadingService: LoaderService)


            In your fetch data function.

            this.loadingService.start('your-purpose');
            this.yourApiService.getSomething()
            .subscribe(something =>
                this.loadingService.stop('your-purpose');
            );


For the buttons to work with async data you have to use observable pipe and tap the the current observable data once the data is fetched you can stop

Example : 


            Inject into constructor LoadingService.

            constructor(private loadingService: LoaderService)


            In your fetch data function.

            this.loadingService.start('your-purpose');
            this.something = this.yourApiService.getSomething()
            .pipe(tap(() => this.loadingService.stop('your-purpose')));


Example : 


            Code used : 

                <loader-button [butonTitle]="'Loader Button'" [componentClass]="'btn btn-primary'" [type]="'submit'" [purpose]="purpose" (btnClick)="simulateRequestOnClick()"></loader-button>

                <loader-button [butonTitle]="'Loader Button'" [componentClass]="'btn btn-primary'" [spinner]="true" [type]="'submit'" [purpose]="purpose2" (btnClick)="simulateRequestOnClick2()"></loader-button>
