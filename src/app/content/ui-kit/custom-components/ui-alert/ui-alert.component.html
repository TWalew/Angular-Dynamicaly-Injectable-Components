Alert Component


Purpose :


The alert component is used to display notifications and errors in the bottom right of the screen.



Usage :


Custom service: constructor(private alertService: AlertService)


To use the alert properly you have to pass some parameters to it's chosen method.


this.alertService.success(1, 2)


  • 1. (string) - This property is used to set the title inside the alert.

  • 2. (string) - This property is used to set the text inside the alert..


IMPORTANT!


Alerts directly work with expression from translation files

Alerts can be success, error, info, warning.

Example : 


            Code used : 

            this.alertService.success('COMMON.SUCCESS', 'This is a success example');

            this.alertService.error('COMMON.ERROR', 'This is a error example');

            this.alertService.warning('Warning!', 'This is a warning example');

            this.alertService.info('Info!', 'This is a info example');
