Custom Components


Loader Button :

The loader button component is used to simulate loading state for the button while the request is finished.


Card Table :

The card table component is used for rendering tables / cards or both.


Banner :

The banner component is used to give to the user information and some action with events.


Tabs :

The tabs component is used to generate a tabs component with dynamic tabs where every tab has it's component.


Slider Button :

The slider button is used as a toggle-able on/off switch.


Confirm Dialog :

The confirm dialog component is used whenever we want a user to confirm an action.


Alert :

The alert component is used to bring up notifications in the bottom right corner.


Circle Progress Bar :

The circle progress bar component is used to display a circle progress svg.


Dialog Header :

The dialog header component is used to display standart header for dialogs with X button for close and title plus optional icon.


Empty Chart :

The empty chart component is used to display standart empty chart.


Heading :

The heading component is used to display standart colored heading.


Progress Bar :

The progress bar component is used to display progress bar horizontal.


Slider :

The slider component is used to display slider bar horizontal.


Timeline :

The timeline component is used to display timeline horizontal.


Widget Loader :

The widget loader is used to simulate the loading state inside widgets.
